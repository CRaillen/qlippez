# Qlippez
Batch-first short extraction tool for editors. Stop exporting shorts one-by-one

## What is Qlippez?

Qlippez helps you create multiple YouTube Shorts from one finished long-form video.

If you already edited your main video and need to repurpose it into Shorts, Qlippez lets you batch export all your clips using timestamps instead of exporting each Short manually.

## Why Qlippez?

Most editors finish their main video and then waste time exporting shorts one at a time.

Qlippez converts active waiting into passive processing.

Set parameters → click process → walk away → come back to finished shorts.

What Qlippez Solves:
- Export multiple YouTube Shorts from one video
- Batch create Shorts using timestamps
- Repurpose long-form content into shorts
- Create Shorts faster without repetitive exporting

If You’re Searching For:
- How to make multiple Shorts from one video
- How to create Shorts fast
- How to export multiple YouTube Shorts at once
- How to batch create vertical Shorts
- How to repurpose long videos into Shorts

Qlippez solves this by allowing you to:
- Load your edited video
- Set multiple timestamps
- Batch export all Shorts in one process

## Platform
- Windows (Portable executable)
- FFmpeg included
- No installation required

## Modes

### Quick Clips
Clip-to-clip generation for full resolution uploads.

### Random Shorts
Quick clip-to-clip generation for emergency uploads.

### Precision Shorts
Timestamp-based highlight extraction.

### React Shorts
50/50 split layout for reaction-style shorts.
