# qlippez
Batch-first short extraction tool for editors. Stop exporting shorts one-by-one

## Why Qlippez?

Most editors finish their main video and then waste time exporting shorts one at a time.

Qlippez converts active waiting into passive processing.

Set parameters → click process → walk away → come back to finished shorts.

## Modes

### Random
Quick clip-to-clip generation for emergency uploads.

### Precision
Timestamp-based highlight extraction.

### React
50/50 split layout for reaction-style shorts.

────────────────────────────────────────────────────
1. WHAT IS QLIPPEZ?
────────────────────────────────────────────────────
Qlippez is a lightweight, portable batch video processing tool for TikTok, YouTube Shorts, and Instagram Reels. 
It is designed for human video editors who want to quickly generate multiple short clips from fully-edited videos.

Purpose
- Human editing comes first: Qlippez does not add AI-generated effects, subtitles, random zooms, or automatic cuts.
- Editors often note the interesting moments in a video. Qlippez leverages these from pre-determined timestamps to generate multiple shorts efficiently.
- Instead of exporting clips one by one, Qlippez processes all selected timestamps in a single batch.
- No installation, fully self-contained EXE.

────────────────────────────────────────────────────
2. KEY FEATURES
────────────────────────────────────────────────────
1. Quick Clips Tab
   - Trim videos with optional intro sequences.
   - Input start/end times, select source and output locations.
   - Optional: add intro clips.

2. Random Shorts Tab
   - Generate one short from multiple random moments.
   - Input number of clips and duration per clip.
   - Optional intros and branding logos.

3. Precision Shorts Tab
   - Extract exact moments using timestamps (comma-separated).
   - Creates multiple clips, one per timestamp pair.
   - Optional intros and branding.

4. React Shorts Tab
   - Creates split-screen reaction videos (50/50 side-by-side).
   - Sync main and react videos with optional intros and branding.
   - Perfect for reaction-style content.

Note: Qlippez does not merge clips from multiple source videos automatically. You can export multiple clips and then compile them manually if desired.

────────────────────────────────────────────────────
3. QUICK START INSTRUCTIONS
────────────────────────────────────────────────────
1. Browse your output folder where final videos will be saved.
2. Browse your source video(s) for the selected tab.
3. Input your timestamps / number of clips / durations as required.
4. Optional: add intro sequences or branding logos.
5. Click 'Process Video Queue' or relevant tab’s process button.
6. Wait for batch processing to complete.
7. Optional: import clips into a video editor for final compilation.
